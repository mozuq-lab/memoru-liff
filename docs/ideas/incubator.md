# Memoru LIFF 機能インキュベータ

## 目的

まだ `backlog` に載せるには広すぎるが、将来の差別化候補として保持したい機能テーマを整理する。

この文書は、複数のブレスト原文から重複する方向性をまとめ直した「中間整理メモ」である。詳細な発想過程や個別の原文は `docs/ideas/archive/` を参照する。

## 1. AI が教える・問い返す学習体験

### 代表案

- ~~AI チューター~~ → **実装済み**（TutorPage: フリートーク / クイズ / 弱点克服の 3 モード）
- 逆 SRS / Teach to Learn
- Rubber Duck Review
- 3秒説明チャレンジ
- 記憶の相棒（AI Buddy） — AI が「クラスメート」として並走し、一緒に悩んだり励ましたりする（`wild.md` より昇格）
- 精緻化プロンプト（Elaborative Interrogation） — 正答後に「なぜ？」を問い、AI が説明を評価。精緻化履歴をカードに紐付けて次回復習時に表示（`archive/agent-team-brainstorm-2026-03-08.md` より。Dr.ミナト × Prof.柊 が独立に提案）
- 文脈交替復習（Contextual Variation Review） — 同一カードを毎回異なる文脈で出題。AI が提示角度を動的変換し、文脈依存記憶からの脱却を促す（`archive/agent-team-brainstorm-2026-03-08.md` より）

### 価値

- 暗記だけでなく「説明できるか」を測れる
- LINE / チャット UI と相性が良い
- 既存の Bedrock 活用と自然につながる

### コメント

AI チューターの基本形は実装済み。今後は「逆 SRS / Teach to Learn」「3秒説明チャレンジ」など、ユーザーがアウトプットする方向の拡張が差別化の次の一手になる。精緻化プロンプトは Agent Team ブレストで2名が独立に提案した共鳴ポイントであり、学習科学のエビデンスも強い。

## 3. メタ認知・弱点分析

### 代表案

- 忘却カジノ
- 自信ギャップ可視化
- 判断較正トレーナー（Calibration Trainer） — 復習前に自信度を3段階で申告、実際の正誤と照合して較正スコアを算出。メタ認知的モニタリング精度を訓練する（`archive/agent-team-brainstorm-2026-03-08.md` より。「自信ギャップ可視化」の発展形）
- つまずきメモ自動収集
- 混同検知モード
- 忘却速度ランキング
- 記憶の外部装置化（Exocortex Sync） — LINE で「今○○どのくらい覚えてる？」と聞くと SRS データから定着率を推定して回答する脳のモニタリング機能（`archive/agent-team-brainstorm-2026-03-08.md` より）

### 価値

- 「覚えていない」だけでなく「なぜ覚えられないか」を扱える
- 既存の review データを再利用しやすい
- 学習効率と継続率の両方に効く

### コメント

実装コストに対して価値が高いテーマ群。大きな新規 UI や外部連携なしで試しやすい。

## 4. 継続支援・通知最適化

### 代表案

- 復習ストリーク＆休息バランサー
- Smart Nudge
- スキマ時間フィット
- 予定連動 SRS
- 学習リズム発見
- ニュース連動型・記憶術（News-Link Mnemonics） — 学習キーワードがニュースに登場した際に LINE で通知し、エピソード記憶と結びつける（`wild.md` より昇格）
- 未来の自分への手紙（Future Self Continuity Letter） — 学習の節目に「未来の自分」へメッセージを記録、期日に LINE 通知で成長データと共に届く。内発的動機づけ強化（`archive/agent-team-brainstorm-2026-03-08.md` より）
- 記憶の時間逆行（Tenet Protocol） — 曜日別の学習完了率パターンから「明日はサボりそう」を予測し、前倒し復習を提案するプロアクティブ SRS（`archive/agent-team-brainstorm-2026-03-08.md` より）

### 価値

- 継続しづらい SRS アプリの弱点に直接効く
- LINE 通知との相性が良い
- 比較的小さな MVP で検証しやすい

### コメント

プロダクトの継続率改善に直結する。派手さは弱いが、事業的には重要度が高い。

## 5. 成長実感・ゲーミフィケーション

### 代表案

- ゴースト復習
- カード熟成度バッジ
- Learning Journey / 記憶のタイムラプス
- Chain Bomb Review
- 記憶コンボチェイン（Recall Combo Chain） — 復習中の連続正答でコンボ蓄積、倍率ボーナス、視覚エフェクト、記憶力ランク昇格。インフラ変更ゼロで実装可能（`archive/agent-team-brainstorm-2026-03-08.md` より）
- 記憶の討伐クエスト（Memory Raid Boss） — 毎週出現するボスを正答ダメージで討伐。苦手カードほど高倍率。FOMO + 締切効果（`archive/agent-team-brainstorm-2026-03-08.md` より）
- デイリーミッションボード（Daily Mission Board） — AI が弱点データを分析して毎日3ミッション自動生成。パーフェクトデイ / ウィークリー報酬で多層エンゲージメント（`archive/agent-team-brainstorm-2026-03-08.md` より）
- 認知負荷アダプター（Cognitive Load Adapter） — セッション中の応答パターンから認知負荷を推定し、出題形式を動的調整。「フロー度」可視化（`archive/agent-team-brainstorm-2026-03-08.md` より）

### 価値

- 継続の心理報酬を増やせる
- 既存レビュー画面に拡張しやすい
- 「勉強感」を弱めて習慣化しやすい

### コメント

比較的実装しやすい案が多い。短期の実験候補として扱いやすい。特にコンボチェインは実装コスト最小で復習体験を即座に改善できる最有力候補。

## 9. LINE プラットフォーム活用

### 代表案

- リッチメニュー学習ダッシュボード（Rich Menu Learning Hub） — per-user Rich Menu を学習状態に応じて動的切替。未復習数・ストリーク・次回復習予定をトーク画面下部に常駐表示（`archive/agent-team-brainstorm-2026-03-08.md` より）

### 価値

- トーク画面を開くだけで学習状態が目に入り、習慣化に直結
- per-user Rich Menu は LINE 公式アカウント独自の機能
- 既存の通知基盤（notification_service / due_push_handler）と自然に連携

### コメント

Rich Menu の動的画像生成（Pillow 等）が必要だが、アーキテクチャ変更は小さい。SRS アプリにとって最重要な「毎日開く理由」を LINE のネイティブ機能で実現できる。
