## 各 Agent の改善提案ハイライト

### KIRA の改善案（ゲーム性を注入）
- **精緻化プロンプト → 「解説チャレンジボーナス」**: AI がS/A/B/C採点、Sランクで獲得EXP3倍
- **文脈交替復習 → 「変異モンスター」**: 同じカードが属性違いで出現、全属性制覇でボーナス
- **判断較正 → 「オールイン」**: 自信度に応じてコインを賭けられるリスク＆リワード
- **Exocortex → 「脳力スカウター」**: 他プレイヤーとの比較「この分野、上位12%」

### Prof.柊 の改善案（学術的裏付けを補強）
- **コンボチェイン**: 困難なカードほど高倍率 + 不正答でもコンボ維持の「チャレンジシールド」で desirable difficulties を促進
- **Raid Boss**: HPを「SRSで期日到来したカードの蓄積量」と連動させ、最適間隔を破壊しない設計に
- **Exocortex × 判断較正の統合**: 問い合わせ前に「自分はどのくらい覚えていると思うか」を先に答えさせ、予測→フィードバックのJOLループに
- **記憶の感染**: 苦戦情報だけでなく「克服した方法」をセットで提示し、構造的解説の共有に

### Dr.ミナト の改善案（心理的リスクを緩和）
- **コンボチェイン**: 断絶時に「記憶の残響」として直前5連続を保存、Loss Aversionの痛みを緩和
- **デイリーミッション**: 3ミッションをゴルディロックス配分（確実/ちょうど良い/背伸び）で固定
- **グループバトル → 協力型**: 競争→全員の学習量合算で共通目標達成、所属欲求を満たす
- **Tenet**: 予測根拠をユーザーに透明開示し、Reactance（押し付けへの反発）を防止

### ユキ の改善案（LINE機能を最大活用）
- **認知負荷アダプター**: クイックリプライのボタン数を負荷に応じて動的変更（高負荷=2択、低負荷=自由入力）
- **コンボチェイン**: ランク昇格時にリッチメニューのデザインが変化、LINE内で「成長した感」を常時可視化
- **精緻化プロンプト**: 音声メッセージ入力対応 → Whisper文字起こし → AI評価で、通勤中ハンズフリー学習
- **Tenet Protocol**: リッチメニュー自動切替APIで「学習チャンスタイム」にメニューデザインが変わりワンタップ復習

## 全 Agent 合意の「最優先3機能」

5人の第1位を集計すると、**3つの機能が複数 Agent の推薦で重複**している:

| 機能 | 推薦 Agent 数 | 推薦者 |
|------|-------------|--------|
| **デイリーミッションボード** | 5/5 | KIRA, 柊, ミナト, ユキ, Neon 全員 |
| **精緻化プロンプト** | 4/5 | 柊, ミナト, Neon, (KIRA は統合案で言及) |
| **判断較正トレーナー** | 3/5 | ミナト, Neon(α), (KIRA は「オールイン」で発展) |

### コンセンサス: 最初に実装すべき「コア3機能」

1. **デイリーミッションボード** — 全 Agent が第1位に含めた唯一の機能
2. **精緻化プロンプト** — 2名が独立に提案し、4名が推薦
3. **判断較正トレーナー** or **コンボチェイン** — セッション体験の強化（教育重視 or ゲーム重視で選択）

この3機能を組み合わせた「認知ループ・トライアングル」(Dr.ミナト命名) が、Phase 2 の結論として最も支持された方向性。
