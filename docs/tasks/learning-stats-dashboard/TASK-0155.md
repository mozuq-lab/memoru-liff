## タスク概要

StatsPage をアプリケーションのナビゲーションとルーティングに統合する。ナビゲーションバーの Settings タブを Stats タブに置き換え、App.tsx に /stats ルートを追加し、HomePage に設定画面への歯車アイコンを追加する。

## 主要実装項目

### 1. Navigation.tsx 修正 🔵

**信頼性**: 🔵 *設計文書 architecture.md のナビゲーション設計・既存 Navigation.tsx より*

ナビゲーションバーの Settings タブを Stats タブに置き換える:
- アイコンを統計アイコン（グラフ等）に変更
- リンク先を /stats に変更
- ラベルを「統計」に変更

**実装ファイル**: `frontend/src/components/Navigation.tsx`

### 2. App.tsx ルーティング追加 🔵

**信頼性**: 🔵 *既存 App.tsx のルーティングパターンより*

React Router v7 のルート定義に /stats パスを追加し、StatsPage コンポーネントをレンダリングする。

**実装ファイル**: `frontend/src/App.tsx`

### 3. HomePage.tsx 歯車アイコン追加 🔵

**信頼性**: 🔵 *設計文書 architecture.md のホーム画面設計より*

HomePage のヘッダー部分に設定画面（/settings）へのリンクとなる歯車アイコンを追加する。ナビゲーションから Settings タブが削除されるため、ここからアクセスできるようにする。

**実装ファイル**: `frontend/src/pages/HomePage.tsx`
