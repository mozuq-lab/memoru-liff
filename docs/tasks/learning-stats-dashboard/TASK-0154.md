## 主要実装項目

### 1. StatsPage ページコンポーネント 🔵

**信頼性**: 🔵 *設計文書 architecture.md のページ構成より*

useStats フックを呼び出し、各セクションコンポーネントにデータを渡すメインページ。ローディング・エラー・空状態のハンドリングを含む。

**実装ファイル**: `frontend/src/pages/StatsPage.tsx`

### 2. StatsSummary + StatCard コンポーネント 🔵

**信頼性**: 🔵 *設計文書 architecture.md のコンポーネント設計より*

学習統計のサマリーカード群を表示するコンポーネント:
- `StatsSummary` - カード群をグリッドレイアウトで配置
- `StatCard` - 個別の統計値カード（アイコン、ラベル、値）

**実装ファイル**: `frontend/src/components/stats/StatsSummary.tsx`, `frontend/src/components/stats/StatCard.tsx`

### 3. ProgressBar + StreakDisplay コンポーネント 🔵

**信頼性**: 🔵 *設計文書 architecture.md のコンポーネント設計より*

- `ProgressBar` - 学習進捗をバー表示（習得済み/学習中/未学習）
- `StreakDisplay` - 連続学習日数の表示

**実装ファイル**: `frontend/src/components/stats/ProgressBar.tsx`, `frontend/src/components/stats/StreakDisplay.tsx`

### 4. WeakCardsList + WeakCardItem コンポーネント 🔵

**信頼性**: 🔵 *設計文書 architecture.md のコンポーネント設計より*

- `WeakCardsList` - 弱点カード一覧のコンテナ
- `WeakCardItem` - 個別の弱点カード表示（質問、正答率、復習回数）

**実装ファイル**: `frontend/src/components/stats/WeakCardsList.tsx`, `frontend/src/components/stats/WeakCardItem.tsx`

### 5. ReviewForecast + ForecastBar コンポーネント 🔵

**信頼性**: 🔵 *設計文書 architecture.md のコンポーネント設計より*

- `ReviewForecast` - 復習予測セクションのコンテナ
- `ForecastBar` - 日別の復習予定カード数をバー表示

**実装ファイル**: `frontend/src/components/stats/ReviewForecast.tsx`, `frontend/src/components/stats/ForecastBar.tsx`
