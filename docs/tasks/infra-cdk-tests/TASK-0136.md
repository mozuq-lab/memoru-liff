## 基本的なテスト要件

### Snapshot テスト 🔵

**信頼性**: 🔵 *REQ-001, REQ-002 より*

- dev Props で synth した CloudFormation テンプレートが Snapshot と一致
- prod Props で synth した CloudFormation テンプレートが Snapshot と一致

### Validation テスト 🔵

**信頼性**: 🔵 *REQ-004 より*

- domainName 指定時に certificateArn 未指定でエラー
- domainName 未指定時はエラーなし

### Fine-grained assertions 🔵

**信頼性**: 🔵 *REQ-003, REQ-005 より*

- prod のみ LogBucket が存在（S3 Bucket 数の検証）
- S3 バケットの BlockPublicAccess が BLOCK_ALL
- S3 バケットの暗号化が S3_MANAGED
- CloudFront のセキュリティヘッダー（HSTS, CSP, X-Content-Type-Options）
- キャッシュ設定（/assets/* 長期キャッシュ, /index.html キャッシュ無効）

## 注意事項

- CloudFront セキュリティヘッダーの検証は `AWS::CloudFront::ResponseHeadersPolicy` リソースに対して行う
- キャッシュポリシーの検証は `AWS::CloudFront::CachePolicy` リソースと `AWS::CloudFront::Distribution` の `CacheBehaviors` で行う
- S3 の BlockPublicAccess は `PublicAccessBlockConfiguration` プロパティで検証
