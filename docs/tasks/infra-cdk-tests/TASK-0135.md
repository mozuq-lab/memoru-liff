## 基本的なテスト要件

### Snapshot テスト 🔵

**信頼性**: 🔵 *REQ-001, REQ-002 より*

- dev Props で synth した CloudFormation テンプレートが Snapshot と一致
- prod Props で synth した CloudFormation テンプレートが Snapshot と一致

### Validation テスト 🔵

**信頼性**: 🔵 *REQ-004 より*

- prod 環境で certificateArn 未指定時にエラーが throw される

### Fine-grained assertions 🔵

**信頼性**: 🔵 *REQ-003, REQ-005 より*

- VPC の NAT Gateway 数が dev=0, prod=1
- RDS の MultiAZ が dev=false, prod=true
- RDS の DeletionProtection が dev=false, prod=true
- ECS の AssignPublicIp が dev=ENABLED, prod=DISABLED
- LogGroup の RemovalPolicy が dev=Delete, prod=Retain
- RDS の StorageEncrypted=true, PubliclyAccessible=false

## 注意事項

- KeycloakStack は VPC, RDS, ECS 等の大量リソースを生成するため、synth に時間がかかる場合がある
- NAT Gateway 数は CloudFormation テンプレート内の `AWS::EC2::NatGateway` リソース数で検証する（`resourceCountIs`）
- ECS の `AssignPublicIp` は `AWS::ECS::Service` の `NetworkConfiguration` 内にある
