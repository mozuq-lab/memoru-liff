## タスク概要

`POST /cards/refine` および `POST /cards/generate` ハンドラーで、`json_body` が `None`（`body: "null"` → `json.loads("null")` → `None`）を返すケースに対応する。`isinstance(body, dict)` チェックを追加し、dict でない場合は 400 エラーを返す。
