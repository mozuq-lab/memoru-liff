## 実装詳細

### 1. ReviewProgress コンポーネント 🔵

**信頼性**: 🔵 *architecture.md のコンポーネント設計・interfaces.ts より*

```typescript
interface ReviewProgressProps {
  current: number;  // 現在のカード番号（1始まり）
  total: number;    // 全体枚数
}
```

- テキスト: `{current} / {total}`
- プログレスバー: `(current / total) * 100` % の幅
- Tailwind CSS でスタイリング（bg-blue-500 等）

**実装ファイル**: `frontend/src/components/ReviewProgress.tsx`

### 2. プログレスバースタイリング 🟡

**信頼性**: 🟡 *UIデザインから妥当な推測*

- バー全体: `bg-gray-200 rounded-full h-2`
- 進捗部分: `bg-blue-500 rounded-full h-2` + width をインラインスタイルで設定
- テキスト: `text-sm text-gray-600`
