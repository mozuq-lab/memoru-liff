## 実装詳細

### 1. フリップアニメーション CSS 🔵

**信頼性**: 🔵 *architecture.md の CSS 実装方針より*

フリップアニメーション用の CSS ファイルを作成する。

**実装ファイル**: `frontend/src/styles/flip-card.css`

```css
.flip-card {
  perspective: 1000px;
}
.flip-card-inner {
  transition: transform 0.4s;
  transform-style: preserve-3d;
  position: relative;
  width: 100%;
  height: 100%;
}
.flip-card-inner.flipped {
  transform: rotateY(180deg);
}
.flip-card-front,
.flip-card-back {
  backface-visibility: hidden;
  position: absolute;
  width: 100%;
  height: 100%;
}
.flip-card-back {
  transform: rotateY(180deg);
}

@media (prefers-reduced-motion: reduce) {
  .flip-card-inner {
    transition: none;
  }
}
```

### 2. FlipCard コンポーネント 🔵

**信頼性**: 🔵 *architecture.md のコンポーネント設計・interfaces.ts より*

FlipCardProps に従ったコンポーネントを実装する。

**実装ファイル**: `frontend/src/components/FlipCard.tsx`

```typescript
interface FlipCardProps {
  front: string;        // 表面テキスト（質問）
  back: string;         // 裏面テキスト（解答）
  isFlipped: boolean;   // フリップ状態
  onFlip: () => void;   // フリップハンドラ
}
```

- カード全体をタップ可能にする（onClick ハンドラ）
- isFlipped に応じて CSS クラスを切り替え
- 表面・裏面のテキストを表示
- Tailwind CSS でカードスタイリング（角丸、影、パディング）
- aria-label でアクセシビリティ対応
