## タスク概要

`deck_service.py` の `update_deck` メソッドで、`description` と `color` の null 送信と未送信を区別できない問題を修正する。TASK-0085 で導入した Sentinel パターン（`_UNSET`）を `update_deck` にも適用し、null → REMOVE、未送信 → 変更なし、値 → SET を区別する。
