## タスク概要

`CardDetailPage` でデッキ変更を保存した後に `DecksContext.fetchDecks()` を呼び出し、デッキの `card_count` / `due_count` を更新する。これにより DecksPage やホーム画面の DeckSummary に正確なカウントが反映される。
