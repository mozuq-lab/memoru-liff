## タスク概要

`review_service.py` の `get_due_cards` メソッドで、`total_due_count` が `limit` パラメータで切り詰められた件数を返す問題を修正する。limit 適用前の全復習対象カード数を `total_due_count` として返すようにする。

## 主要実装項目

### total_due_count 修正 🔵

**信頼性**: 🔵 *architecture.md セクション5・dataflow.md フロー5 より*

- `total_due_count = len(all_due_cards)` を limit 前に計算
