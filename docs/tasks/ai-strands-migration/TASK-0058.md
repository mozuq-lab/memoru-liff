## 実装詳細

### 1. API レスポンス形式の完全一致検証 🔵
**信頼性**: 🔵 *api-endpoints.md より確定*

POST /cards/generate エンドポイントが USE_STRANDS フラグの値に関わらず同一の GenerateCardsResponse を返すことを検証する。

**検証項目**:
- HTTP ステータスコード: 200 OK
- レスポンス本体の JSON スキーマが一致
- generated_cards 配列の要素構造が一致
- 各カード要素（CardGenerated DTO）が以下を含む:
  - id: カードID（UUID）
  - question: 問題文
  - answer: 正答
  - explanation: 説明
  - difficulty: 難易度
  - tags: タグ配列
- generation_info が以下を含む:
  - model_used: 使用モデル名（値は異なってよい：claude-3-5-sonnet vs claude-3-opus など）
  - generation_tokens: 使用トークン数
  - processing_time_ms: 処理時間

**実装ファイル**: `backend/tests/unit/test_migration_compat.py`（新規）

### 2. エラーハンドリング互換性検証 🔵
**信頼性**: 🔵 *api-endpoints.md より確定*

エラーシナリオにおいて両方のサービスが同一の HTTP ステータスコードとエラーレスポンス形式を返すことを検証する。

**検証対象エラーケース**:

| シナリオ | Expected Status | Expected Error Code |
|--------|-----------------|-------------------|
| Agent タイムアウト | 504 | GENERATION_TIMEOUT |
| レート制限超過 | 429 | RATE_LIMITED |
| JSON 解析エラー | 500 | PARSE_ERROR |
| 内部エラー | 500 | INTERNAL_ERROR |
| 入力バリデーション失敗 | 400 | VALIDATION_ERROR |
| ユーザー未認証 | 401 | UNAUTHORIZED |
| ユーザー未登録 | 404 | USER_NOT_FOUND |

**エラーレスポンス形式**:
```json
{
  "error": {
    "code": "ERROR_CODE",
    "message": "Error message",
    "details": {
      "field": "additional context"
    }
  }
}
```

**実装ファイル**: `backend/tests/unit/test_migration_compat.py`

### 4. フィーチャーフラグ切替テスト 🔵
**信頼性**: 🔵 *architecture.md より確定*

USE_STRANDS 環境変数の動的な切替機能と、フラグ未設定時のデフォルト動作を検証する。

**テストシナリオ1: USE_STRANDS=true での切替**
```
Given: USE_STRANDS=true, StrandsAIService がデプロイされている
When: POST /cards/generate を呼び出す
Then:
  - AIService Factory が StrandsAIService を返す
  - Strands Agent が呼び出される
  - GenerateCardsResponse が返される
```

**テストシナリオ2: USE_STRANDS=false での切替**
```
Given: USE_STRANDS=false, BedrockAIService がデプロイされている
When: POST /cards/generate を呼び出す
Then:
  - AIService Factory が BedrockAIService を返す
  - Bedrock InvokeModel が呼び出される
  - GenerateCardsResponse が返される
```

**テストシナリオ3: フラグ未設定時のデフォルト動作**
```
Given: USE_STRANDS 環境変数が未設定
When: アプリケーションが起動する
Then:
  - 無設定時のデフォルト値（false または true）が適用される
  - architecture.md で指定されたデフォルト値が使用される
```

**環境変数の確認**:
- USE_STRANDS: "true" | "false"
- デフォルト値は architecture.md で確定すること

**実装ファイル**: `backend/tests/unit/test_migration_compat.py`
