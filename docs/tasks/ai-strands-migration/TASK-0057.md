### 4. generate_cards() メソッド実装 🔵
**信頼性**: 🔵 *dataflow.md より確定*

Strands Agent を使用してカード生成を実行する。

**メソッドシグネチャ**:
```python
def generate_cards(
    self,
    topic: str,
    difficulty: str = "intermediate",
    card_count: int = 5,
    language: str = "ja"
) -> GenerationResult:
```

**処理フロー**:
1. prompts/generate.py から ユーザープロンプトを生成
   - topic, difficulty, card_count, language を使用
2. Strands Agent を呼び出し（system_prompt + user_prompt）
   - 呼び出し形式: `agent(user_prompt)` を想定（🟡 SDK 調査時に確認）
3. レスポンスを受け取る
4. _parse_generation_result() で GenerationResult に変換
5. GenerationResult を返却

**エラーハンドリング**:
- Agent タイムアウト（Strands SDK exception）→ AITimeoutError
- Provider エラー（Bedrock/Ollama エラー）→ AIProviderError
- JSON パース失敗 → AIParseError

**実装ファイル**: `backend/src/services/strands_service.py`

### 5. _parse_generation_result() メソッド実装 🔵
**信頼性**: 🔵 *interfaces.py より確定*

Strands Agent のレスポンスを GenerationResult DTO に変換する。

**メソッドシグネチャ**:
```python
def _parse_generation_result(self, response: dict) -> GenerationResult:
```

**処理フロー**:
1. response から "cards" 配列を抽出
2. 各カード要素を CardGenerated DTO に変換
   - question, answer, explanation フィールド抽出
   - 形式検証（必須フィールドの存在確認）
3. GenerationResult を構築
   - cards: List[CardGenerated]
   - generation_info: GenerationInfo（model_used, generation_tokens など）
4. GenerationResult を返却

**例外処理**:
- JSON 構造不正 → AIParseError with 詳細メッセージ
- 必須フィールド欠落 → AIParseError with フィールド名

**実装ファイル**: `backend/src/services/strands_service.py`
