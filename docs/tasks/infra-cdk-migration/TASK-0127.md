## 主要実装項目

### 1. CognitoStackProps インターフェース定義 🔵

**信頼性**: 🔵 *設計文書 architecture.md 環境管理パターンより*

```typescript
interface CognitoStackProps extends cdk.StackProps {
  environment: 'dev' | 'staging' | 'prod';
  cognitoDomainPrefix: string;
  callbackUrls: string[];
  logoutUrls: string[];
}
```

**実装ファイル**: `infrastructure/cdk/lib/cognito-stack.ts`

### 2. UserPool 定義 🔵

**信頼性**: 🔵 *既存テンプレート・設計文書 architecture.md Cognito セクションより*

- `selfSignUpEnabled: true`
- `signInAliases: { email: true }`
- `mfa: Mfa.OPTIONAL`, `mfaSecondFactor: { sms: false, otp: true }`
- パスワードポリシー: 8文字以上、大文字・小文字・数字・記号必須
- `accountRecovery: EMAIL_ONLY`
- `deletionProtection: isProd`

**実装ファイル**: `infrastructure/cdk/lib/cognito-stack.ts`

### 3. UserPoolClient 定義 🔵

**信頼性**: 🔵 *既存テンプレート・設計文書より*

- `generateSecret: false` (Public client for SPA + PKCE)
- OAuth: `authorizationCodeGrant: true`, スコープ: OPENID, PROFILE, EMAIL
- トークン有効期限: access/id 1時間、refresh 30日
- `preventUserExistenceErrors: true`

**実装ファイル**: `infrastructure/cdk/lib/cognito-stack.ts`

### 4. CfnOutput 定義 🔵

**信頼性**: 🔵 *設計文書 architecture.md Outputs セクションより*

| Output | Export 名 | 値 |
|---|---|---|
| UserPoolId | `memoru-{env}-cognito-user-pool-id` | User Pool ID |
| UserPoolClientId | `memoru-{env}-cognito-client-id` | Client ID |
| OidcIssuerUrl | `memoru-{env}-cognito-oidc-issuer` | OIDC Issuer URL |
| CognitoDomainUrl | `memoru-{env}-cognito-domain-url` | Cognito ドメイン URL |

**実装ファイル**: `infrastructure/cdk/lib/cognito-stack.ts`
