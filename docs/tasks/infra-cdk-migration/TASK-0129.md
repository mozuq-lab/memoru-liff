interface KeycloakStackProps extends cdk.StackProps {
  environment: 'dev' | 'staging' | 'prod';

## 注意事項

- `ApplicationLoadBalancedFargateService` で ALB, TargetGroup, Listener, ECS Service, SecurityGroup, LogGroup が自動生成される（既存テンプレートの約300行相当）
- RDS の SecurityGroup は別途定義し、ECS タスクからの接続を許可する必要がある
- prod 環境では `certificateArn` 未指定時に TypeScript レベルでエラーを投げる
