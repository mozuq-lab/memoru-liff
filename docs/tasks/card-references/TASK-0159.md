## 主要実装項目

### 1. TypeScript 型定義追加 🔵

**信頼性**: 🔵 *バックエンド Reference モデルと対応*

**実装ファイル**: `frontend/src/types/card.ts`

```typescript
export type ReferenceType = "url" | "book" | "note";

export interface Reference {
  type: ReferenceType;
  value: string;
  label?: string;
}
```

既存の `Card`, `CreateCardRequest`, `UpdateCardRequest`, `CardResponse` 型にも `references` フィールドを追加:

```typescript
export interface Card {
  // ... 既存フィールド ...
  references: Reference[];
}

export interface CreateCardRequest {
  // ... 既存フィールド ...
  references?: Reference[];
}

export interface UpdateCardRequest {
  // ... 既存フィールド ...
  references?: Reference[];
}
```

### 2. ReferenceEditor コンポーネント 🔵

**信頼性**: 🔵 *設計文書のコンポーネント仕様より*

**実装ファイル**: `frontend/src/components/ReferenceEditor.tsx`

Props:
```typescript
interface ReferenceEditorProps {
  references: Reference[];
  onChange: (references: Reference[]) => void;
}
```

機能:
- 「参考情報を追加」ボタンで新しい参考情報行を追加
- 各行にタイプセレクター（ドロップダウン: URL / 書籍 / メモ）
- 各行に value 入力フィールド（テキスト入力）
- 各行に label 入力フィールド（Optional、テキスト入力）
- 各行に削除ボタン
- onChange コールバックで親に変更を通知

### 3. ReferenceDisplay コンポーネント 🔵

**信頼性**: 🔵 *設計文書のコンポーネント仕様より*

**実装ファイル**: `frontend/src/components/ReferenceDisplay.tsx`

Props:
```typescript
interface ReferenceDisplayProps {
  references: Reference[];
}
```

機能:
- references が空の場合は何も表示しない
- URL タイプ: クリック可能なリンク（`<a href={value} target="_blank" rel="noopener noreferrer">`）、label があれば label を表示テキストに使用
- book タイプ: 書籍アイコン + テキスト表示
- note タイプ: メモアイコン + テキスト表示
