## 主要実装項目

### 1. Reference Pydantic モデル追加 🔵

**信頼性**: 🔵 *要件定義書・設計文書より*

**実装ファイル**: `backend/src/models/card.py`

Reference モデルを新規追加する:

```python
from enum import Enum

class ReferenceType(str, Enum):
    URL = "url"
    BOOK = "book"
    NOTE = "note"

class Reference(BaseModel):
    type: ReferenceType
    value: str
    label: str | None = None
```

- `type`: 参考情報の種類（URL / 書籍 / メモ）
- `value`: URL、書名、メモ内容など
- `label`: 表示ラベル（Optional、未指定時は value をそのまま表示）

### 4. DynamoDB シリアライズ/デシリアライズ 🔵

**信頼性**: 🔵 *既存 to_dynamodb_item/from_dynamodb_item パターンに準拠*
