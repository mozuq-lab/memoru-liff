## タスク概要

CardService の `create_card()` と `update_card()` を拡張し、参考情報（references）の作成・更新に対応する。既存カード（references フィールドなし）との後方互換性を保証する。

- **前提タスク**: [TASK-0157: Reference モデル + Card モデル拡張](TASK-0157.md)
- **後続タスク**: なし

- [x] `card_service.create_card()` で references 省略時は空リストで作成される
- [x] `card_service.update_card()` が references を更新できる
- [x] `card_service.update_card()` で references 未指定時は既存値を維持する

- [x] 全テストがパスしている

## 主要実装項目

### 1. create_card() の references 対応 🔵

**信頼性**: 🔵 *既存 create_card パターンに準拠*

**実装ファイル**: `backend/src/services/card_service.py`

CreateCardRequest の references フィールドを Card 生成時に渡す:

```python
def create_card(self, request: CreateCardRequest, user_id: str) -> Card:
    card = Card(
        # ... 既存フィールド ...
        references=request.references,
    )
    # ... DynamoDB 保存 ...
```

### 2. update_card() の references 対応 🔵

**信頼性**: 🔵 *既存 update_card パターンに準拠*

**実装ファイル**: `backend/src/services/card_service.py`

UpdateCardRequest の references が指定されている場合のみ更新する:

```python
def update_card(self, card_id: str, request: UpdateCardRequest, user_id: str) -> Card:
    card = self.get_card(card_id, user_id)
    # ... 既存フィールド更新 ...
    if request.references is not None:
        card.references = request.references
    # ... DynamoDB 保存 ...
```

### 3. 後方互換性の保証 🔵

**信頼性**: 🔵 *from_dynamodb_item のデフォルト値で対応済み（TASK-0157）*

TASK-0157 で `from_dynamodb_item()` に `data.get("references", [])` を実装済みのため、サービス層では特別な対応不要。既存カードの取得時に自動的に空リストが返る。

## 基本的なテスト要件

### create_card テスト 🔵

**信頼性**: 🔵 *既存テストパターンに準拠*

**テストファイル**: `backend/tests/services/test_card_service.py`

- references 付きの CreateCardRequest でカード作成 → 保存されたカードに references が含まれる
- references 省略の CreateCardRequest でカード作成 → 空リストで保存される

### update_card テスト 🔵

**信頼性**: 🔵 *既存テストパターンに準拠*

- references を指定して update → references が更新される
- references を None（未指定）で update → 既存の references が維持される
- references を空リストで update → references がクリアされる

### 後方互換性テスト 🔵

**信頼性**: 🔵 *DynamoDB データパターンより*

- references フィールドのない既存 DynamoDB アイテムから Card を復元 → references が空リスト
- 復元した Card の to_response() で references が空リストとして返る
