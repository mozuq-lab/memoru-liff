```python
class GenerateStudyNotesRequest(BaseModel):
    source_type: Literal["deck", "tag"] = Field(..., description="生成ソース種別")
    source_id: str = Field(..., description="デッキID or タグ名", min_length=1, max_length=200)
```

**実装ファイル**: `backend/src/models/study_notes.py`
