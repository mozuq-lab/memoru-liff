### 1. StrandsAIService 実装 🔵

**信頼性**: 🔵 *既存StrandsAIService実装パターン（grade_answer等）より*

```python
def generate_study_notes(self, cards: List[CardForNotes]) -> StudyNotesResult:
    with self._handle_ai_errors():
        start = time.time()
        prompt = build_study_notes_prompt(cards)
        agent = Agent(model=self.model, system_prompt=prompt)
        result = agent(user_message)
        content = str(result)
        elapsed = int((time.time() - start) * 1000)
        return StudyNotesResult(
            content=content,
            model_used=self.model_id,
            processing_time_ms=elapsed,
        )
```

**実装ファイル**: `backend/src/services/strands_service.py`
