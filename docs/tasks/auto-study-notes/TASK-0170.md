## タスク概要

既存の `CardService` のカード作成・更新・削除処理に、study-notes キャッシュの無効化ロジック（is_stale=true更新）を追加する。カードの deck_id と tags に紐づくキャッシュを無効化する。カード更新時にdeck_idやtagsが変更された場合は、旧値のキャッシュも無効化する。

- **後続タスク**: [TASK-0175: バックエンド統合テスト](TASK-0175.md)

## 完了条件

- [ ] カード作成時に deck_id のキャッシュが無効化される
- [ ] カード作成時に tags のキャッシュが無効化される
- [ ] カード更新時に deck_id のキャッシュが無効化される
- [ ] カード更新時に deck_id が変更された場合、旧デッキのキャッシュも無効化される
- [ ] カード更新時に tags が変更された場合、旧タグのキャッシュも無効化される
- [ ] カード削除時に deck_id のキャッシュが無効化される
- [ ] キャッシュが存在しない場合でもエラーにならない
- [ ] 既存のカードCRUDテストが引き続き通る
- [ ] 新規テストが通る
