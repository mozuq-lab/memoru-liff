## 実装詳細

### 1. StudyNotesService クラス定義 🔵

**信頼性**: 🔵 *architecture.md・既存サービスパターンより*

```python
class StudyNotesService:
    def __init__(self, study_notes_table=None):
        self.table = study_notes_table or boto3.resource("dynamodb").Table(
            os.environ["STUDY_NOTES_TABLE"]
        )
```

**実装ファイル**: `backend/src/services/study_notes_service.py`

### 2. get_cached メソッド 🔵

**信頼性**: 🔵 *REQ-ASN-023・database-schema.md より*

DynamoDB GetItem で `user_id` + `source_key` をキーにキャッシュを取得。存在しない場合は None を返す。

### 3. save_cache メソッド 🔵

**信頼性**: 🔵 *REQ-ASN-021・database-schema.md より*

DynamoDB PutItem でキャッシュを保存。is_stale=false, generated_at=現在時刻。

### 4. invalidate_cache メソッド 🔵

**信頼性**: 🔵 *REQ-ASN-022・設計ヒアリング「同期的フラグ更新」より*

DynamoDB UpdateItem で is_stale=true に更新。ConditionExpression で存在チェック。
