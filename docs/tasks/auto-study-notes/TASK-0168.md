## 実装詳細

### 1. generate_or_get メソッド 🔵

**信頼性**: 🔵 *dataflow.md フロー1・フロー2より*

```python
def generate_or_get(self, user_id, source_type, source_id):
    cached = self.get_cached(user_id, source_type, source_id)
    if cached and not cached.is_stale:
        return cached

    if source_type == "deck":
        cards = self.get_cards_for_deck(user_id, source_id)
    else:
        cards = self.get_cards_for_tag(user_id, source_id)

    result = self.ai_service.generate_study_notes(cards)
    self.save_cache(user_id, source_type, source_id, result, len(cards))
    return ...
```

**実装ファイル**: `backend/src/services/study_notes_service.py`
