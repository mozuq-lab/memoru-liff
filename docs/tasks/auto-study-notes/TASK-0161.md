## 実装詳細

### 1. SAM テンプレートにテーブル定義を追加 🔵

**信頼性**: 🔵 *database-schema.md・既存SAMテンプレートパターンより*

`backend/template.yaml` の Resources セクションに以下を追加:

```yaml
StudyNotesTable:
  Type: AWS::DynamoDB::Table
  Properties:
    TableName: !Sub "${AWS::StackName}-study-notes"
    BillingMode: PAY_PER_REQUEST
    AttributeDefinitions:
      - AttributeName: user_id
        AttributeType: S
      - AttributeName: source_key
        AttributeType: S
    KeySchema:
      - AttributeName: user_id
        KeyType: HASH
      - AttributeName: source_key
        KeyType: RANGE
    PointInTimeRecoverySpecification:
      PointInTimeRecoveryEnabled: true
```

**実装ファイル**: `backend/template.yaml`

### 2. Lambda環境変数の追加 🔵

**信頼性**: 🔵 *既存テーブル設定パターンより*

既存のLambda関数（ApiFunction）の環境変数に `STUDY_NOTES_TABLE` を追加。

### 3. IAMポリシーの更新 🔵

**信頼性**: 🔵 *既存IAMポリシーパターンより*

Lambda関数のIAMポリシーに study-notes テーブルへの GetItem, PutItem, UpdateItem, Query 権限を追加。
