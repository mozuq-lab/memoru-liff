## 実装詳細

### 1. プロンプトテンプレート関数 🟡

**信頼性**: 🟡 *REQ-ASN-031〜034の要件内容は確定だが、プロンプトの文言は推測*

```python
def build_study_notes_prompt(cards: list[CardForNotes]) -> str:
    """要約ノート生成用プロンプトを構築する."""
    sanitized_cards = [_sanitize_card(c) for c in cards]
    cards_text = _format_cards(sanitized_cards)
    return STUDY_NOTES_SYSTEM_PROMPT.format(cards=cards_text)
```

**実装ファイル**: `backend/src/services/prompts/study_notes.py`

### 2. カード内容のサニタイズ 🔵

**信頼性**: 🔵 *NFR-ASN-102・既存プロンプトインジェクション対策より*

カードの front/back から制御文字やプロンプト指示を除去する処理。

### 3. 言語自動判定 🟡

**信頼性**: 🟡 *REQ-ASN-051から妥当な推測。判定ロジックの詳細は未確定*

カード内容の言語を検出し、出力言語をプロンプトに含める。
