## タスク概要

handler.py の _get_user_id_from_event と shared.py の get_user_id_from_context の JWT dev フォールバックロジックを shared.py に統一。ENVIRONMENT=dev AND AWS_SAM_LOCAL=true の二重条件に強化。フォールバック発動時に logger.warning を出力。

## 依存タスク

- **前提タスク**: なし
- **後続タスク**: [TASK-0111: 構造化ログ移行](TASK-0111.md)

- [ ] handler.py の _get_user_id_from_event を削除し shared.py の get_user_id_from_context に統一
- [ ] AWS_SAM_LOCAL 環境変数チェックを追加
- [ ] フォールバック発動時の logger.warning を実装
- [ ] テストカバレッジ 80% 以上を維持

## 主要実装項目

### 1. shared.py の get_user_id_from_context に環境変数二重チェック追加 🔵
