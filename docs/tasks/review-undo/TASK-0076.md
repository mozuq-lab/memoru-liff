復習完了画面（ReviewComplete）を大幅修正し、セッション結果一覧を表示する。各カード行を表示する`ReviewResultItem`コンポーネントを新規作成する。カード表面テキスト、グレード（色分け）、次回復習日、取り消しボタンを表示する。

## 主要実装項目

### 1. ReviewComplete大幅修正 🔵

**信頼性**: 🔵 *architecture.md ReviewComplete修正セクション・interfaces.ts ReviewCompletePropsより*

既存の`reviewedCount`のみの表示から、`results`配列を受け取って結果一覧を表示するよう変更:
- props: `{ results, onUndo, isUndoing, undoingIndex }`
- 結果一覧をスクロール可能なリストで表示
- 「ホームに戻る」ボタンは維持

**実装ファイル**: `frontend/src/components/ReviewComplete.tsx`

### 2. ReviewResultItem新規作成 🟡

**信頼性**: 🟡 *architecture.md ReviewResultItemセクション・interfaces.ts ReviewResultItemPropsから妥当な推測*

各結果行を表示するコンポーネント:
- カード表面テキスト（長文時はtext-overflow: ellipsis で省略）
- グレード表示（`GRADE_DISPLAY_CONFIGS`に基づく色分け）
- 次回復習日表示
- 取り消しボタン（type === 'graded' の場合のみ表示）
- undo中のローディング表示（isUndoing && 該当index の場合）

**実装ファイル**: `frontend/src/components/ReviewResultItem.tsx`

### 3. グレード色分け定数 🟡

**信頼性**: 🟡 *interfaces.ts GRADE_DISPLAY_CONFIGSから妥当な推測*

既存GradeButtons.tsxの色スキームと統一したグレード表示設定を定義。

**実装ファイル**: `frontend/src/components/ReviewResultItem.tsx`（または共通定数ファイル）
