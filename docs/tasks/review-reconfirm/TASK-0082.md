## 主要実装項目

### 1. GradeButtons再確認モード対応 🔵

**信頼性**: 🔵 *要件定義書 REQ-002, REQ-102・設計文書より*

**実装ファイル**: `frontend/src/components/GradeButtons.tsx`

- 新しいprops追加: `isReconfirmMode`, `onReconfirmRemembered`, `onReconfirmForgotten`
- `isReconfirmMode = true` の場合:
  - 6段階評価ボタンを非表示
  - 「覚えた」「覚えていない」ボタンを表示
  - スキップボタンを非表示
- `isReconfirmMode = false` の場合: 従来通りの表示
- 「覚えた」ボタン: 緑系の色（成功イメージ）
- 「覚えていない」ボタン: 赤/オレンジ系の色

### 2. ReconfirmBadge新規コンポーネント 🔵

**信頼性**: 🔵 *要件定義書 REQ-101・ヒアリングQ1回答より*

**実装ファイル**: `frontend/src/components/ReconfirmBadge.tsx`

- 「再確認」テキストのバッジ表示
- Tailwindでスタイリング（背景色で目立たせる）
- ReviewPage内で`isReconfirmMode`が`true`の場合のみ表示

### 3. ReviewPage UIの統合 🔵

**信頼性**: 🔵 *設計文書 architecture.md・データフロー図より*

**実装ファイル**: `frontend/src/pages/ReviewPage.tsx`

- ReconfirmBadgeの条件付き表示
- GradeButtonsにisReconfirmMode、onReconfirmRemembered、onReconfirmForgottenを渡す
- 再確認カード表示時のフリップカード表示（表面→裏面→ボタン、通常と同じフロー）

### 4. ReviewComplete / ReviewResultItem拡張 🔵

**信頼性**: 🔵 *要件定義書 REQ-501・ヒアリングQ3回答より*

**実装ファイル**:
- `frontend/src/components/ReviewComplete.tsx`
- `frontend/src/components/ReviewResultItem.tsx`

- `type: 'reconfirmed'` の結果表示:
  - 元の評価（quality値）のバッジを色付きで表示
  - 「覚えた✔」のサブラベルを追加
- Undoボタンは再確認カードでも表示（REQ-404）
- 通常カード（grade 3-5）の表示は従来通り変更なし
