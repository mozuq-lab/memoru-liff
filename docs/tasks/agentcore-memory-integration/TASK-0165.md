### 1. _create_agent() の改修 🔵

**信頼性**: 🔵 *設計文書 architecture.md StrandsTutorAIService 改修設計セクションより*

**実装ファイル**: `backend/src/services/tutor_ai_service.py`

```python
def _create_agent(self, system_prompt: str, session_manager=None, messages=None):
    from strands import Agent
    kwargs = {
        "model": self.model,
        "system_prompt": system_prompt,
    }
    if session_manager:
        kwargs["session_manager"] = session_manager
        kwargs["agent_id"] = "tutor"
    elif messages:
        kwargs["messages"] = messages
    return Agent(**kwargs)
```

### 2. generate_response() の改修 🔵

```python
def generate_response(self, system_prompt, messages_or_user_message, session_manager=None):
    if session_manager:
        # SessionManager モード: 単一メッセージを受け取る
        user_message = messages_or_user_message
        agent = self._create_agent(system_prompt, session_manager=session_manager)
        response = agent(user_message)
    else:
        # 後方互換モード: 全履歴を受け取る（既存動作）
        messages = messages_or_user_message
        # 既存ロジック維持
        ...
```
