## 実装詳細

### 1. パラメータ追加 🔵

**信頼性**: 🔵 *設計文書 architecture.md SAMテンプレート変更セクションより*

```yaml
Parameters:
  TutorSessionBackend:
    Type: String
    Default: "agentcore"
    AllowedValues:
      - "agentcore"
      - "dynamodb"
    Description: Tutor session history backend (agentcore or dynamodb)

  AgentCoreMemoryId:
    Type: String
    Default: ""
    Description: AgentCore Memory ID for tutor session management
```

**実装ファイル**: `backend/template.yaml`

### 2. 環境変数追加 🔵

**信頼性**: 🔵 *REQ-005・設計文書より*

Globals.Function.Environment.Variables に追加:

```yaml
TUTOR_SESSION_BACKEND: !Ref TutorSessionBackend
AGENTCORE_MEMORY_ID: !Ref AgentCoreMemoryId
```

### 3. IAM ポリシー追加 🟡

**信頼性**: 🟡 *REQ-402・AgentCore Memory API アクセスに必要な IAM アクション（具体的なアクション名は SDK ドキュメント確認が必要）*

Lambda 実行ロールの Policies に AgentCore Memory API アクセス権限を追加:

```yaml
- Statement:
    - Effect: Allow
      Action:
        - "bedrock-agentcore:*"
      Resource: "*"
```

### 4. SAM テンプレートの検証 🔵

**信頼性**: 🔵 *既存開発フローより*

```bash
cd backend && sam validate
```
