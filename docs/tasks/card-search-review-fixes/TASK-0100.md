## 主要実装項目

### FilterChips 条件付き表示 🔵

**信頼性**: 🔵 *設計文書 architecture.md セクション1・要件定義書 REQ-001 より*

`CardsPage.tsx` で `activeTab !== "due"` の条件レンダリングを追加する。

```tsx
{activeTab !== "due" && (
  <FilterChips value={reviewStatus} onChange={setReviewStatus} />
)}
```

**実装ファイル**: `frontend/src/pages/CardsPage.tsx`

### SortSelect 型ガード導入 🟡

**信頼性**: 🟡 *設計文書 architecture.md セクション4・要件定義書 REQ-101 より（TypeScript ベストプラクティスから妥当な推測）*

`e.target.value as SortByOption` を型ガード関数に置き換える。

```ts
const isSortByOption = (value: string): value is SortByOption =>
  (["created_at", "next_review_at", "ease_factor"] as string[]).includes(value);
```

**実装ファイル**: `frontend/src/components/SortSelect.tsx`

### FilterChips role 属性改善 🟡

**信頼性**: 🟡 *設計文書 architecture.md セクション5・要件定義書 REQ-103 より（WAI-ARIA ベストプラクティスから妥当な推測）*

`role="group"` → `role="radiogroup"` に変更（排他的選択のセマンティクス）。

**実装ファイル**: `frontend/src/components/FilterChips.tsx`
