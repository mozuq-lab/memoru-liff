## 主要実装項目

### normalize 関数の共通化 🔵

**信頼性**: 🔵 *設計文書 architecture.md セクション3・要件定義書 REQ-102 より*

`frontend/src/utils/text.ts` を新規作成し、`normalize` 関数を移動する。`useCardSearch.ts` と `HighlightText.tsx` からローカル定義を削除し、共通ユーティリティからインポートする。

- `frontend/src/hooks/useCardSearch.ts`（修正）

### reset 関数のメモ化 🔵

**信頼性**: 🔵 *設計文書 architecture.md セクション2・要件定義書 REQ-002 より*

`reset` 関数を `useCallback` でラップする。依存配列は空 `[]`（setState 関数は安定参照のため）。

**実装ファイル**: `frontend/src/hooks/useCardSearch.ts`

### デッドコード削除 🔵

**信頼性**: 🔵 *要件定義書 REQ-005 より*
