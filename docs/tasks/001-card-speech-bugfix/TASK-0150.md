**信頼性**: 🔵 *WAI-ARIA switch パターン + architecture.md REQ-101 より*

**実装ファイル**: `frontend/src/pages/SettingsPage.tsx`

```tsx
// L292 付近
<button
  type="button"
  role="switch"
  aria-checked={speechSettings.autoPlay}
  aria-label="自動読み上げ"  // ← 追加
  data-testid="autoplay-toggle"
  onClick={() =>
    updateSpeechSettings({ autoPlay: !speechSettings.autoPlay })
  }
  ...
>
```
