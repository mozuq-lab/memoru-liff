## 主要実装項目

### 1. ReviewPage speechProps 修正（3箇所） 🔵

**信頼性**: 🔵 *architecture.md REQ-001 設計方針より*

**実装ファイル**: `frontend/src/pages/ReviewPage.tsx`

3箇所の `speechProps` オブジェクト生成を以下のパターンに修正:

```tsx
// 修正前（通常モード、L664-667）
speechProps={{
  speechState: { isSpeaking, isSupported },
  onSpeakFront: () => speak(currentCard.front),
  onSpeakBack: () => speak(currentCard.back),
}}

// 修正後
speechProps={{
  speechState: { isSpeaking, isSupported },
  onSpeakFront: () => (isSpeaking ? cancel() : speak(currentCard.front)),
  onSpeakBack: () => (isSpeaking ? cancel() : speak(currentCard.back)),
}}
```

同様の修正を以下にも適用:
- 再採点モード（L534-537）: `regradeCard.front` / `regradeCard.back`
- 再確認モード（L601-604）: `reconfirmCard.front` / `reconfirmCard.back`
